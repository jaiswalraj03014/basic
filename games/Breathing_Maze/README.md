# 🎵 Breathing Maze - Calm Rhythm Game

## Overview
Breathing Maze is a therapeutic rhythm-based game that combines mindful breathing patterns with interactive maze navigation. Players guide a glowing orb through mazes by tapping in sync with carefully designed rhythm patterns, promoting relaxation and cognitive focus.

## 🎮 How to Play

### Basic Controls
- **Tap/Touch**: Tap your screen or press spacebar in rhythm with the visual/audio cues
- **Timing**: Your taps must align with the rhythm pattern to move the orb forward
- **Objective**: Navigate the orb through the maze by maintaining rhythm accuracy

### Game Mechanics
1. **Rhythm Patterns**: Each level features different rhythm patterns (4/4 time, syncopated, triplets, etc.)
2. **Tolerance Window**: Timing accuracy tolerance decreases as levels progress (200ms → 50ms)
3. **Movement**: Successful taps move the orb forward through the maze
4. **Progression**: Complete mazes to advance to more complex rhythm patterns

### Levels & Difficulty
- **Level 1**: Simple 4/4 rhythm (60 BPM)
- **Level 2**: Mixed rhythms with varied note lengths
- **Level 3**: Syncopated patterns with complex timing
- **Level 4**: Triplet patterns requiring precise timing
- **Level 5+**: Dynamic variations with randomized timing

## 🚀 Getting Started

### Installation
1. Navigate to the Breathing_Maze directory
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the game: `python app.py`
4. Open your browser to `http://localhost:8000`

### Tips for Success
- Find a quiet, comfortable space
- Use headphones for better audio feedback
- Start with slower rhythms and build up
- Focus on your breathing while playing
- Take breaks if you feel frustrated
