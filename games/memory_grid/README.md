# 🧠 Memory Grid - Working Memory Training Game

## Overview
Memory Grid is a scientifically-designed working memory training game that challenges players to remember and reproduce increasingly complex sequences of grid positions. This game is specifically designed to enhance working memory capacity, attention control, and cognitive flexibility through adaptive difficulty progression.

## 🎮 How to Play

### Basic Controls
- **Click/Tap**: Click on grid positions to reproduce the shown sequence
- **Visual Memory**: Watch the sequence of highlighted grid positions
- **Reproduction**: Click the same positions in the correct order
- **Audio Cues**: Listen to sound patterns that correspond to positions

### Game Mechanics
1. **Sequence Display**: Watch as grid positions light up in a specific order
2. **Memory Challenge**: Remember the exact sequence of positions
3. **Reproduction Phase**: Click the positions in the same order
4. **Adaptive Difficulty**: Sequence length increases with successful completions
5. **Performance Feedback**: Receive detailed analysis of your cognitive performance

### Levels & Difficulty
- **Level 1-2**: 3-4 position sequences (beginner)
- **Level 3-4**: 5-6 position sequences (intermediate)
- **Level 5-6**: 7-8 position sequences (advanced)
- **Level 7+**: 9-12 position sequences (expert)
- **Adaptive Scaling**: Difficulty adjusts based on your success rate

## 🚀 Getting Started

### Installation
1. Navigate to the memory_grid directory
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the game: `python main.py`
4. Open your browser to `http://localhost:8000`

### Tips for Success
- Find a quiet, distraction-free environment
- Use headphones for better audio cues
- Focus on accuracy before attempting speed
- Take breaks if you feel mental fatigue
- Practice daily for optimal results
