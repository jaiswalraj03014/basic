# 🌈 Color Maze - Visual Processing & Cognitive Training Game

## Overview
Color Maze is a dynamic visual processing game that challenges players to navigate through color-coded mazes while making split-second decisions based on visual cues. The game enhances visual attention, pattern recognition, and decision-making speed through engaging maze-based gameplay.

## 🎮 How to Play

### Basic Controls
- **Movement**: Use arrow keys or WASD to navigate through the maze
- **Color Recognition**: Follow color-coded paths and avoid obstacles
- **Speed Challenge**: Navigate quickly while maintaining accuracy
- **Pattern Following**: Recognize and follow visual patterns

### Game Mechanics
1. **Color-Coded Navigation**: Different colors represent different paths and obstacles
2. **Time Pressure**: Complete mazes within time limits for bonus points
3. **Visual Cues**: Pay attention to color changes and visual indicators
4. **Progressive Difficulty**: Maze complexity increases with each level

### Levels & Difficulty
- **Level 1**: Simple 2-color maze with clear paths
- **Level 2**: 3-color maze with branching paths
- **Level 3**: Complex multi-color maze with time pressure
- **Level 4**: Dynamic color-changing obstacles
- **Level 5+**: Advanced pattern recognition challenges

## 🚀 Getting Started

### Installation
1. Navigate to the color_maze directory
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the game: `python game.py`
4. Open your browser to `http://localhost:8000`

### Tips for Success
- Ensure good lighting for optimal color perception
- Use a comfortable viewing distance
- Take breaks to avoid eye strain
- Practice with different color combinations
- Focus on accuracy before speed
