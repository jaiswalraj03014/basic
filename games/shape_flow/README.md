# 🔄 Shape Flow - Pattern Recognition & Cognitive Flexibility Game

## Overview
Shape Flow is a dynamic pattern recognition game that challenges players to identify and match shapes and colors in rapid succession. The game enhances cognitive flexibility, visual processing speed, and decision-making abilities through engaging shape-based challenges that adapt to your skill level.

## 🎮 How to Play

### Basic Controls
- **Click/Tap**: Select the correct shape or color from the active slots
- **Pattern Recognition**: Identify the next piece in the sequence
- **Quick Decision Making**: Make choices rapidly as pieces flow through
- **Life System**: Start with 3 lives, lose one for each incorrect choice

### Game Mechanics
1. **Pattern Display**: Watch as shapes/colors appear in the queue
2. **Active Slots**: Choose from 3 active slots that match the pattern
3. **Flow Control**: Select the correct piece to continue the sequence
4. **Score System**: Earn 10 points for each correct choice
5. **Life Management**: Maintain lives to continue playing

### Game Levels
- **Level 1 - Shape Flow**: Match circle, square, and triangle patterns
- **Level 2 - Color Sync**: Match red, blue, and green color sequences
- **Future Levels**: Advanced pattern combinations and faster speeds

## 🚀 Getting Started

### Installation
1. Navigate to the shape_flow directory
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the game: `python app.py`
4. Open your browser to `http://localhost:8000`

### Tips for Success
- Focus on accuracy before attempting speed
- Look for patterns in the sequence flow
- Use peripheral vision to track multiple elements
- Take breaks if you feel mental fatigue
- Practice regularly for optimal results
