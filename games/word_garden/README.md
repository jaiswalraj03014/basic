# 🌸 Word Garden - Vocabulary Building & Language Learning Game

## Overview
Word Garden is an innovative vocabulary-building game that combines language learning with visual rewards. Players answer vocabulary questions to grow and nurture a beautiful virtual garden, creating an engaging and motivating environment for expanding vocabulary, improving language skills, and enhancing cognitive-linguistic abilities.

## 🎮 How to Play

### Basic Controls
- **Multiple Choice**: Select the correct answer from four options
- **Vocabulary Questions**: Answer questions about word meanings, synonyms, and definitions
- **Garden Growth**: Correct answers help your garden plants grow and flourish
- **Streak System**: Build consecutive correct answers for enhanced plant growth

### Game Mechanics
1. **Question Display**: Read vocabulary questions with multiple choice answers
2. **Answer Selection**: Choose the correct definition or synonym
3. **Plant Growth**: Correct answers plant seeds and help existing plants grow
4. **Garden Progression**: Watch your garden evolve from seeds to beautiful flowers
5. **Streak Rewards**: Consecutive correct answers accelerate plant growth

### Garden Growth Stages
- **Seed Stage**: First correct answer plants a new seed
- **Sprout Stage**: Second correct answer grows the plant
- **Flower Stage**: Third correct answer produces a beautiful flower
- **Rich Flower Stage**: Fourth correct answer creates an enhanced bloom

## 🚀 Getting Started

### Installation
1. Navigate to the word_garden directory
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the game: `python app.py`
4. Open your browser to `http://localhost:8000`

- Read questions carefully before selecting answers
